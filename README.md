# Proyecto-Pokemon

Este proyecto es una aplicación web sencilla que permite:

- 🔎 Buscar Pokémon por nombre usando la [PokéAPI](https://pokeapi.co/).
- ⭐ Guardar tus Pokémon favoritos en el navegador utilizando **localStorage**.
- 📋 Mostrar una lista de favoritos persistente, incluso al recargar la página.

## Funcionalidades

1. Buscar Pokémon
   - Ingresa el nombre de un Pokémon en el campo de búsqueda.
   - Se muestra su nombre e imagen oficial.

2. Guardar favoritos
   - Puedes añadir el Pokémon buscado a tu lista de favoritos.
   - Los favoritos se almacenan en "localStorage" para que no se pierdan al cerrar la página.

3. Lista de favoritos
   - Se renderiza automáticamente al cargar la página.
   - Cada Pokémon favorito se muestra con su nombre e imagen.

##  Tecnologías utilizadas

- HTML5 → estructura básica de la aplicación.
- CSS3 :estilos simples para presentación.
- JavaScript
  - "fetch()"  para consumir la PokéAPI.
  - "localStorage" para persistencia de datos.
  - "JSON.stringify()" y "JSON.parse()" para manejar arrays/objetos.
